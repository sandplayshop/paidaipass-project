# AI 應用規劃師初級題庫練習（智能 AI 題庫版・個人使用）

這是一個針對 **iPAS「AI 應用規劃師初級」** 考試設計的線上刷題工具。  
整個系統使用純前端技術（HTML + CSS + JavaScript），並支援 PWA，可安裝到桌面或手機主畫面，離線也能使用。

> 本專案為**個人練習版本**，題庫為 AI 生成模擬題，**非官方題庫**。

---

## 功能總覽

### 1. 主題庫（`index.html`）

- 題庫練習頁面，支援多種練習模式：
  - 隨機練習
  - 順序練習
  - 錯題複習
  - 收藏複習
- 題目來源：`new_questions.json`
- 題目結構包含：
  - 題號、題目、4 個選項、答案
  - 分類（七大主題）
  - 解析（支援 Markdown 的粗體、換行）
  - 難度（基礎／中級／進階）
  - 關鍵字（方便未來擴充用）

### 2. 學習儀表板（`dashboard.html`）

- 可視化顯示練習歷史與學習成效：
  - 每日練習題數
  - 正確／錯誤／未作答統計
  - 各分類的正確率
- 使用 `localStorage` 保存歷史練習紀錄，資料只存在使用者本機瀏覽器。

> **注意：** 儀表板是讀取 `index.html` 練習過程中所寫入的 `localStorage.quizHistory` 資料。

## 授權機制

為了限制本工具為「個人授權使用」，在 `index.html` 中加入了 **授權 Email 驗證機制**：

- 一開始只會看到「授權 Email 輸入區」。
- 只有輸入 **指定 Email** 才會解鎖「題庫練習設定畫面」。
